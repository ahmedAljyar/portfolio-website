خطوات إنشاء المشروع
1- قسمت الصفحة إلي قمسين الرأس (header) والمحتوي (.content)
2- وخصصت للرأس طول 12% ووضعت فيه الشعار (.logo) "AHMED" وقائمة التنقل (nav) تتنقل بين ثلاثة أجزاء رئيسية (المشاريع, عن, تواصل)
3- ثم خصصت للجزء الخاص بالمحتوي باقي الطول إلا 3% للمسافة بينه وبين الرأس بواسطة (margin) ووضعت له الخاصية (overflow: auto) لكي نقدر علي التمرير لرؤية باقي المحوي
4- ووضعت في جزء المحتوي رسالة ترحيبية والثلاثة أقسام الأساسية التي خصصت لها أزرار التنقل ووضعت لهم الفئة class حتي أخصص لهم بعض إضافات css (.section) وأيضا id لكل قسم لكي يساعد علي التنقل
5- ووضعت في كل قسم منهم عنوان رئيسي ووضعت له الفئة (.sectionName) لخصائص css
6- وفي قسم المشاريع وضعت فيه حاوية للبطاقات (.cards) وكل بطاقة (.card) عبارة عن مشروع, تنقسم البطاقة إلي الجزء العلوي والذي يحتوي علي صورة للمشروع وجزء التفاصيل الذي يحتوي علي اسم المشروع ووصفه
7- ووضعت لحاوية البطاقات (المشاريع) الخاصيتين (display: flex, flex-wrap: wrap) حتي تأتي المشاريع تحت بعضها إن لم يكن هناك مساحة كافية
8- وبواسطة java script وضعت حدث النقر علي كل بطاقة مشروع لكي يظهر نافذة popup يكون فيها تفاصيل المشروع كاملة لأن البطاقة تقص جزءا من الوصف أو الاسم لو كانا طويلين وأيضا فيها صورة المشروع وتحتها زر لزيارة المشروع
9- ثم في قسم عن كتبت بعض الأشياء عني وفي قسم تواصل كتبت رقم هاتف غير حقيقي وبعض الروابط
